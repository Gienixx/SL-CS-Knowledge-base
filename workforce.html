Invite User

Initial permissions

The user receives a secure invitation link and appears immediately with an Invited badge and generated SL- ID. No temporary password is shown.

Edit User

Effective permissions

Name and email changes are synchronized across Supabase Auth, the user profile, compatibility login, and identity link. If synchronization fails, the previous values are restored. Attendance correction and approval permissions apply only to administrators. Payroll access does not include either permission.

Create Schedule

Choose how many days you want to load.

Create Schedule Entry

Published schedules are visible to the assigned agent. Scheduled entries remain drafts. To enable repetition, first add exactly one entry for every day from Sunday through Saturday, including rest days, then check the box while saving the final entry. Admin edits to generated schedules are not overwritten.
