Invite User

Initial permissions

The user receives a secure invitation link, then creates their password after accepting it. They appear immediately with an Invited badge and generated SL- ID.

Edit User

Effective permissions

Name and email changes are synchronized across Supabase Auth, the user profile, compatibility login, and identity link. If synchronization fails, the previous values are restored. Payroll permissions are granted individually and are not included with administrator access. Payroll access does not include either permission.

Schedule settings
